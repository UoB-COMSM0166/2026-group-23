**User Story1(来自 Player)**

As a player,

I want to play a ball-throwing mini-game to earn coins

So that l can use those coins to upgrade my towers

**Acceptance Criteria:**

Given l have started a level

When l complete the ball-throwing mini-game,

Then lreceive coins based on my performance.

**User Story 2(来自 Player)**

As a player,

I want to upgrade my towers using coins,

So that l can defend against stronger enemies.

**Acceptance Criteria:**

Given l have enough coins

When l select a tower and choose an upgrade,

Then the tower's attributes (e.g. damage or range) increase.

**User Story3(来自 Game Designer)**

As a game designer,

I want the coin rewards from the mini-game to be balanced

So that the tower defense gameplay remains challenging

**Acceptance Criteria:**

Given a standard level difficulty,

When the mini-game is played with average performance,

Then the coins earned are sufficient but not excessive.
